This post was created 
from the github.com website 
using the pull request system.

Latex equations 
are rendered by `MathJax <http://www.mathjax.org/>`_
and can be inserted like so:

.. math::
    
    (\lambda + \mu)\nabla(\nabla\cdot\mathbf{u}) + \mu\nabla^2\mathbf{u} - \rho \partial_t^2 \mathbf{u} = - \mathbf{f}

or inline like :math:`\mathbf{u} = (u_x, u_y, y_z)`.
