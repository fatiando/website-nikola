Fatiando a Terra (Portuguese for *Slicing the Earth*) is an open-source Python toolkit for geophysical modeling and inversion.

In a wider scope, we also aim to be an open repository for geophysics.

We provide a home for:

  • Open-source software
  • Texts like lecture notes and exercises (free under Creative Commons licenses)
  • A blog for discussion of ideas, recent publications, programming, etc

Development

The Fatiando a Terra software package is under development on GitHub! Check out the source code, docs, and other information. You can also take a look at the functionality already implemented!

If you need any help or want to contribute, get in touch!
